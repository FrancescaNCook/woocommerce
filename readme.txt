Easily integrate MultiSafepay payment solutions into your WooCommerce webshop with this free MultiSafepay WooCommerce plugin.

== Description ==
**About MultiSafepay**

MultiSafepay is a collecting payment service provider which means we take care of the agreements, technical details and
payment collection required for each payment method. You can start selling online today and manage all your transactions
from one place.


**Supported Payment Methods**

[WordPress](https://docs.multisafepay.com/integrations/woocommerce/faq/ \"The supported Payment Methods & Gift cards for this plugin\")


**Requirements**
Wordpress & WooCommerce installed
To use the plugin you need a MultiSafepay account, [WordPress](https://testmerchant.multisafepay.com/signup \"Create a test account\")


== Installation ==
- Download the plugin by clicking the Download button on this page
- Navigate to the backend of your WooCommerce webshop and go to Plugins -> Add New. Click on Browse file
- Upload the Plugin_WooCommerce_x.x.x.zip file.
- Login in your WooCommerce Admin and navigate to WooCommerce -> Settings -> Payments
- Scroll down to the section MultiSafepay settings and configure the plugin. Information about the API key can be found on our API key page
- Navigate to WooCommerce -> Settings -> Checkout -> Payments. Enable the payment methods you would like to offer.
